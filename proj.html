Proyectos

En esta sección se recopilan algunos de los proyectos que a mi parecer son de los más significativos que he realizado.

Detección de Covid-19
Este proyecto tenía la finalidad de alimentar una red neuronal convolucional con radiografías de personas con Covid 19 para después predecirlo. Además de esto, también detectaba si una persona padecía neumonía.

Código en Github

App de figuras
Utilizando OpenGL, esta app fue diseñada para dispositivos Android. El usuario escribe alguna de las figuras disponibles junto con sus parámetros y se muestra en la pantalla. Para leer las figuras y parámetros que el usuario ingresaba, se utilizó un analizador léxico, sintáctico y semántico.

En progreso
En colaboración con el Instituto de Ciencia de Datos, estamos desarrollando una aplicación web que sea de utilidad para los geólogos de diferentes lugares para la clasificación de rocas y análisis de datos.
